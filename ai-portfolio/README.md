# 🚀 Premium AI Engineer Portfolio

A world-class, award-winning single-file portfolio website for AI Engineers.

## ✨ Features

- **Animated particle background** with neural network connections
- **Custom cursor** with glow effect and magnetic hover
- **Typing animation** cycling through roles
- **Glassmorphism UI** with smooth hover states
- **Scroll reveal animations** throughout
- **Animated AI workflow diagram**
- **Tech stack marquee** (dual-row, infinite scroll)
- **Contribution graph** visualization
- **Animated counters** for achievements
- **Working contact form** (UI)
- **Fully responsive** (mobile, tablet, desktop)
- **Loading screen** with progress bar
- **Back to top button**
- **SEO optimized** meta tags

## 🚀 Deploy in 60 Seconds

### Option 1: Netlify (Recommended)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop the `ai-portfolio` folder
3. Done! Live in seconds.

### Option 2: Vercel
1. Go to [vercel.com](https://vercel.com)
2. Import or drag the folder
3. Deploy instantly.

### Option 3: GitHub Pages
1. Create a GitHub repo
2. Upload `index.html`
3. Settings → Pages → Deploy from main branch

### Option 4: Any Web Host
Upload `index.html` to any hosting provider's public folder.

## ✏️ Customization

Open `index.html` and find/replace:

| Placeholder | Replace With |
|-------------|--------------|
| `hello@aiengineer.dev` | Your email |
| `+1 (555) 000-0000` | Your phone |
| `AI.Engineer` | Your name/brand |
| GitHub, LinkedIn links (`href="#"`) | Your actual URLs |
| Resume download alert | Your PDF URL |
| Project cards | Your real projects |
| Testimonials | Real client testimonials |
| Stats/counters | Your real numbers |
